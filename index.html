Scribbletune

Create music with JavaScript & Node.js!

Let's C how this works

Install Scribbletune

npm install scribbletune

Generate a C major scale.

var scribble = require('scribbletune');

var clip = scribble.clip({
    notes: scribble.scale('c', 'major', 3), // root, scale and octave
    pattern: 'x_'.repeat(8) // `x` implies play a note
});

scribble.render(clip, 'cscale.mid');

This will generate a MIDI file called cscale.mid in the same location as you ran the above script. This is how it sounded when I imported it in Garage Band:

Chords

There's more to Scribbletune than generating scales. You can create rhythms and generate chords via Scribbletune.

var scribble = require('scribbletune');
var chords = scribble.clip({
	notes: ['f#m', 'c#m', 'DMaj', 'Bm', 'EMajor', 'AMaj', 'dmaj', 'c#m', 'AMaj'],
	pattern: 'x_x_x_--'.repeat(8),
	sizzle: true
});  
 
scribble.render(chords, 'chords.mid');

Simple beat with a bassline

You can generate multiple MIDI files and import them in your favorite music software and render them as beats or songs. Here we 'll generate a basic 4x4 kick drum loop, a simple bass line and a hi-hats loop.

var scribble = require('scribbletune');
 
// 4 by 4 kick
var kick = scribble.clip({
	notes: ['c4'],
	pattern: 'x---'.repeat(4)
});
 
// A simple bass line
var bass = scribble.clip({
	notes: scribble.scale('c#', 'harmonic major').filter((a, x) => x % 2 === 0),
	pattern: 'xxx_xx_x',
	shuffle: true
});

var hats = scribble.clip({
	notes: ['h4'],
	pattern: 'x'.repeat(16),
	accentMap: 'x---x---x-x---x-
});

// Render the 3 clips to MIDI files
scribble.midi(bass.concat(bass), 'bass.mid');
scribble.midi(kick, 'kick.mid');
scribble.midi(hats, 'hats.mid');

Here's how that sounds when imported into Garage Band and rendered to a mp3 file.

Software

You can use the MIDI files generated via Scribbletune in Ableton Live, Propellerhead Reason, Fruity Loops or any music software that lets you work with MIDI.
